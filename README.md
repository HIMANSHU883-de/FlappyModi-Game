# FlappyModi-Game

A browser-based Flappy Bird clone built using HTML, CSS, and JavaScript. The game features custom-themed assets, obstacle generation, collision detection, score tracking, sound effects, and a game-over system.

## Features

- Gravity-based bird movement
- Dynamic obstacle generation
- Collision detection
- Real-time score tracking
- Sound effects
- Game over and restart functionality
- Built with vanilla JavaScript

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Gameplay

Control the bird using the **Spacebar** and navigate through obstacles while achieving the highest score possible.

## Learning Outcomes

This project demonstrates:

- DOM Manipulation
- Event Handling
- JavaScript Game Loops
- Collision Detection
- Dynamic Element Creation
- Audio Integration
- CSS Positioning and Animations

## How to Run

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/FlappyModi-Game.git
```

2. Open the project folder.

3. Run `index.html` in a browser or use Live Server.
